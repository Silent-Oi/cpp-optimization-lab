# M1 — 单个阻尼振子参考

目标：建立欠阻尼模型的可信标量参考，并确定批量版本使用预计算精确一步更新系数。

## 模型决定

- [x] 在 `design.md` 记录使用 `omega` 和阻尼比 `zeta` 表达参数。
- [x] 第一版限定 `0 < zeta < 1`，并保留 `zeta = 0` 回归情况。
- [x] 明确状态、原始物理参数和派生更新系数的职责。
- [ ] 明确非法参数和浮点输入的处理方式。

## 参考实现

- [ ] 推导或独立核对欠阻尼解析状态公式。
- [ ] 实现任意时刻阻尼解析状态。
- [ ] 推导固定 `dt` 下的精确一步 `2x2` 更新系数。
- [ ] 实现单个振子的预计算系数更新。
- [ ] 保证系数只在参数或 `dt` 改变时重新计算。

## 正确性证据

- [ ] `t = 0` 恢复任意初始状态。
- [ ] 一步更新与解析状态在约定容差内一致。
- [ ] 多步更新与直接计算终止时刻解析状态一致。
- [ ] `zeta = 0` 退化为无阻尼结果。
- [ ] 不同 `omega`、`zeta` 和初始状态均通过。
- [ ] 解析结果的机械能非增。

## 学习证据与完成门槛

- [ ] 能由模型说明为什么阻尼加速度依赖速度。
- [ ] 能说明物理耗散与数值耗散的区别。
- [ ] 能解释为什么预计算精确一步系数适合本项目的批量性能实验。
- [ ] 参考实现、一步更新和回归检查全部可信。

## 可选主题的进入条件

以下内容不计入核心进度：

- **RK4**：需要比较通用高阶积分方法，而不是为了补齐方法列表。
- **完整收敛阶实验**：需要进一步验证积分器理论阶数时再做。
- **手动 SIMD**：自动向量化不足、核心循环仍为计算瓶颈，且复杂度具有学习价值。
- **临界/过阻尼**：后续问题明确需要覆盖不同阻尼区间。
- **线程池**：至少两个项目出现重复线程调度需求后，再考虑进入 Common。
