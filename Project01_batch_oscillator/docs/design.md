## 模型约定

线性黏性阻尼模型：

\[
m x'' + c x' + kx = 0
\]

使用：

\[
\omega=\sqrt{k/m},\qquad
\zeta=\frac{c}{2\sqrt{km}}
\]

得到一阶系统：

\[
\dot{x}=v,\qquad
\dot{v}=-2\zeta\omega v-\omega^2x
\]

第一版参数边界：

- `omega` 有限且严格大于零。
- `zeta` 有限，核心物理范围为 `0 < zeta < 1`。
- `zeta = 0` 仅用于无阻尼回归。
- 所有振子使用统一 `dt`，但可以拥有不同状态、`omega` 和 `zeta`。
- 第一版统一使用 `double`。

## 精确一步更新

对于参数不随时间变化的振子：

\[
y_{n+1}=M(\omega,\zeta,\Delta t)y_n,
\qquad y=[x,v]^T
\]

初始化阶段计算更新矩阵的四个系数，核心循环只进行两个线性组合。

选择这一方案的目的不是提前追求最快实现，而是：

- 使直接解析状态和多步更新具有相同物理模型。
- 使 AoS/SoA 对比不受不同积分误差干扰。
- 明确区分初始化成本和热循环成本。
- 为“更多预计算是否值得更多内存读取”提供后续实验入口。

## 正确性约定

按以下层次建立证据：

1. 无阻尼解析解和已有积分器的最低回归。
2. 阻尼解析状态在 `t = 0` 恢复初值。
3. 精确一步更新与解析状态一致。
4. 多步更新与直接计算终止时刻一致。
5. `zeta = 0` 退化为无阻尼模型。
6. 批量更新与逐个标量更新一致。
7. AoS、SoA 和可选并行版本在约定容差内一致。

连续阻尼模型的机械能应满足非增趋势，但数值检查仍需设置浮点容差，不能用逐步严格小于作为唯一正确性条件。
