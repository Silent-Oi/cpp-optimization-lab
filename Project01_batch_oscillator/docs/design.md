# Project01 设计记录

## 1. 问题定义

Project01 分为两层模型：

- 无阻尼简谐振子：用于学习解析解、数值积分、误差和稳定性。
- 线性黏性阻尼振子：作为后续批量计算和性能实验的最终模型。

在这里记录最终采用的变量含义、参数表达和输入范围。

需要明确：

- 振子数量
- 每个振子的状态
- 固有角频率的含义和范围
- 阻尼参数采用阻尼比还是阻尼系数，以及允许范围
- 每个振子的物理参数是否相同
- 时间步长
- 总时间步数
- 初始条件
- 数值积分方法

## 2. 正确性标准

说明如何判断实现是正确的。

无阻尼学习基线可考虑：

- 与解析解比较
- 能量误差
- 特定初始条件下的周期性

阻尼模型可考虑：

- 与解析解或独立高精度参考结果比较
- 零阻尼时退化为简谐振子结果
- 位移包络和机械能的衰减趋势
- 区分物理耗散与数值方法引入的额外耗散或增能

所有模型都需要记录：

- 不同实现之间的结果差异
- 浮点误差允许范围
