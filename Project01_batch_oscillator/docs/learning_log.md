## 7.18号
1. 辛欧拉，verlet，全部实现。
2. 在我看来三种数值计算方式本质都是泰勒展开的使用，所以文件名改为了 taylor_formula
本质：
显式欧拉: 是x和v分别用一次泰勒展开计算，误差大，能量扩散

辛欧拉: 说我先用一次泰勒展开算一个，比如算 xn+1，算出来之后用 an+1 算 vn+1，
然后这时候发现: 我草转换矩阵 M 的 det 等于 1，为啥会这样呢。
然后在一想，我们操作一共两步，可以用 M1 和 M2 表示，M2 操作位移，M2 = [[1,s1],[0,1]], 用 vn 计算 xn+1，且速度不会变化，行列式为1；
M1 操作速度，M1 = [[1,0],[s2,1]]，用更新后的 xn+1 计算 vn+1，且不会改变 xn+1，行列式为1；
M = M1 * M2，det(M) = 1
这就是辛欧拉能量稳定的本质。
但是为了速度和位移稳定，还需要保证 M 1.有特征值，2.而且还得有俩，俩模长为1。这样 M 作用任意的向量都可以把作用的向量用两个特征向量线性组合，那只要两个特征值模长都为1，向量的两个方向就能保证稳定。
先来看1，有特征向量，则 det(M-λI) = 0，可以得一个已 detM 和 tr(M) 为系数的关于 λ 的二次方程。
再来看2，解要有俩而且都为1，那首先俩解不能直接等于1，这样的 M 作用任何一个向量都不会发生变化了，所以要有虚根 e^±iθ，那二次方程就要满足 tr^2 < 4。
这俩虚根刚好也表示对俩特征向量选择θ。
这样就求得辛欧拉计算过程速度和位移的稳定条件。

verlet：前面跟辛欧拉一样，只是用二次泰勒展开算一个
后面跟辛欧拉有点区别，算另一个不直接用 an+1，而是用 (an+1 + an)/2 这样的平均加速度算 vn+1。
然后呢，唉他最后 M 的 det 还是刚好等于1，这是为啥呢。怎么这平均加速度也能保证能量守恒呢。
那其实 verlet 的操作可以分三步
第一步：M1 = [[1,0],[s1,1]]，算 h/2 的速度，v' = vn + an * h/2 
第二步：M2 = [[1,s2],[0,1]]，根据中间时间的速度算出完整位移 xn+1, xn+1 = xn + v'*h
第三部：M1，再算出 h/2 速度 vn+1 = v‘ + an+1 * h/2
上面每一步的行列式都为1
M(h) = M1(h/2) * M2(h) * M1(h/2)
而且啊 M1(-h) = M1(h)^-1, M2(-h) = M2(h)^-1
那么y(t+h) = M(h) * y(t) => y(t) = M(-h) * y(t+h)
也就是verlet法的操作是对称的，给定末状态可以任意往前推过去状态

上面就是我对三种计算方法的理解，rk4 应该和他们本质不一样吧。
