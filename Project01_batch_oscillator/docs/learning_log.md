## 7.18号
1. 辛欧拉，verlet，全部实现。
2. 在我看来三种数值计算方式本质都是泰勒展开的使用，所以文件名改为了 taylor_formula
本质：
显式欧拉: 是x和v分别用一次泰勒展开计算，误差大，能量扩散

辛欧拉: 说我先用一次泰勒展开算一个，比如算 xn+1，算出来之后用 an+1 算 vn+1，
然后这时候发现: 我草转换矩阵 M 的 det 等于 1，为啥会这样呢。
然后在一想，我们操作一共两步，可以用 M1 和 M2 表示，M2 操作位移，M2 = [[1,s1],[0,1]], 用 vn 计算 xn+1，且速度不会变化，行列式为1；
M1 操作速度，M1 = [[1,0],[s2,1]]，用更新后的 xn+1 计算 vn+1，且不会改变 xn+1，行列式为1；
M = M1 * M2，det(M) = 1
这就是辛欧拉能量稳定的本质。
但是为了速度和位移稳定，还需要保证 M 1.有特征值，2.而且还得有俩，俩模长为1。这样 M 作用任意的向量都可以把作用的向量用两个特征向量线性组合，那只要两个特征值模长都为1，向量的两个方向就能保证稳定。
先来看1，有特征向量，则 det(M-λI) = 0，可以得一个已 detM 和 tr(M) 为系数的关于 λ 的二次方程。
再来看2，解要有俩而且都为1，那首先俩解不能直接等于1，这样的 M 作用任何一个向量都不会发生变化了，所以要有虚根 e^±iθ，那二次方程就要满足 tr^2 < 4。
这俩虚根刚好也表示对俩特征向量选择θ。
这样就求得辛欧拉计算过程速度和位移的稳定条件。

verlet：前面跟辛欧拉一样，只是用二次泰勒展开算一个
后面跟辛欧拉有点区别，算另一个不直接用 an+1，而是用 (an+1 + an)/2 这样的平均加速度算 vn+1。
然后呢，唉他最后 M 的 det 还是刚好等于1，这是为啥呢。怎么这平均加速度也能保证能量守恒呢。
那其实 verlet 的操作可以分三步
第一步：M1 = [[1,0],[s1,1]]，算 h/2 的速度，v' = vn + an * h/2 
第二步：M2 = [[1,s2],[0,1]]，根据中间时间的速度算出完整位移 xn+1, xn+1 = xn + v'*h
第三部：M1，再算出 h/2 速度 vn+1 = v‘ + an+1 * h/2
上面每一步的行列式都为1
M(h) = M1(h/2) * M2(h) * M1(h/2)
而且啊 M1(-h) = M1(h)^-1, M2(-h) = M2(h)^-1
那么y(t+h) = M(h) * y(t) => y(t) = M(-h) * y(t+h)
也就是verlet法的操作是对称的，给定末状态可以任意往前推过去状态

上面就是我对三种计算方法的理解，rk4 应该和他们本质不一样吧。

## 7.23号

1. 欠阻尼振子微分方程解。

我真得说道说道了，这玩意算了一天，终于想明白了，总共三条路。

  1. 直接解x(t), 设x(t) = e^rt, 求解得到俩虚根r1, r2对应俩解x1(t), x2(t)
  因为解的线性组合也是微分方程解，x(t) = c1 x1(t) + c2 x2(t); v(t) = x'(t)
  带入初值条件解出c1和c2。

  2. 直接展开e^At, y' = Ay, y = e^At * y0

  A = [[0,1], [-w^2, -2β]]; B = A + β I = [[β, 1],[-w^2, -β]]
  泰勒展开e^(B-βI)t，结合sin和cos的泰勒展开得到解析解形式。

  3. 太麻烦了我懒得展开了，就是这个算了一天给我算的晕死了，说一下思路
    1. y' = Ay
    2. 设x(t) = e^rt, 则y = e^rt*[1,r]T = e^rt * q
    3. 联立1和2，r刚好是A的特征值, q是特征向量, 求出r1,r2,q1,q2
    4. 把y拆分成线性组合y=a1(t)v1 + a2(t)v2
    5. 用特征向量线性组合y，解得a1(t) a2(t)带未知系数解，结合初值条件解出系数。

2. 演化矩阵M的计算 y(h) = M(h)y(0)。

  方法一 算出来x和v的解析解，系数用初值x0v0表示，转化成关于x0v0的线性方程组

  方法二 直接算出演化矩阵

  方法三 用状态矩阵A替代M。
  P = [q1, q2], Λ = [[r1, 0], [0, r2]];
  A P = P Λ, A = P Λ P-1

  e^At = e^(P-1 Λ P)t = P [[e^r1t, 0], [0, e^r2t]] P-1
  yt = P [[e^r1t, 0], [0, e^r2t]] P-1 * y0

  仔细观察方法三，仔细看其中的形式，然后你就能对线代理解更进一步，得到第四种计算方法

  方法四 基向量e1 = [1,0]T, e2 = [0,1]T, 按列组成E = [e1, e2]
  yt = xt * e1 + vt * e2 = E * [xt, vt]T
  y0 = E * [x0, v0]T

  特征向量q1,q2, 按列组成P=[q1,q2]
  yt = a1t * q1 + a2t * q2 = P * Z(t), Z(t) = [a1t, a2t]T
  根据条件可以直接解得
  a1t = c1 * e^r1t, a2t =c2 * e^r2t 
  yt = c1 * e^r1t * q1 + c2 * e^r2t * q2

  P * [c1, c2] T = y0
  [c1, c2] T = P-1 * y0

  yt = P * [c1 * e^r1t, c2 * e^r2t]T
  yt = P * [[e^r1t, 0], [0, e^r2t]] [c1, c2] T
  yt = P [[e^r1t, 0], [0, e^r2t]] P-1 * y0

  直接记一下结论吧
  方法三的形式可以看出，先把y0转换到P基空间，Z0 = P-1 y0
  然后在P基空间沿着特征向量分布演化，Zt = [a1t, a2t] * Z0
  最后再转回原空间,yt = P * Zt

  然后回忆一下求解时候的方法三，发现线性组合和转化空间有明显关系
  基向量按列组成P，P乘以P空间向量可以把向量转化回原本单位向量E空间，就可以得到方法四了。
  线性组合内含(省略了过程)了空间转化
