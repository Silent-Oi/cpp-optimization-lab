# 项目路线图

## Project00 — Common

定位：仓库公共基础设施。

主要学习内容：

- 工程目录组织
- CMake 基础
- 公共头文件管理
- 测试与 benchmark 分离
- 可复现的实验记录

Project00 会随着后续项目逐步增长，不要求一次完成。

## Project01 — Batch Oscillator

定位：第一个性能优化项目。

主要学习内容：

- 连续内存
- AoS 与 SoA
- 循环组织
- Release 构建
- 基础 benchmark
- 数值正确性与性能并重
