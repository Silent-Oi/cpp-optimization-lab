## 项目完成标准

- [ ] 核心计算结果正确且有独立验证。
- [ ] benchmark 可以在记录的环境中复现。
- [ ] 至少一个优化结论有明确证据。
- [ ] 能解释主要瓶颈和未采用方案的原因。
- [ ] 已记录可迁移到下一项目的经验。
